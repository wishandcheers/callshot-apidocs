# Diff Data

**Context**: comparison | **Type**: aggregate-root

## Definition
Complete structural diff between two API versions, containing per-group (api/admin) raw diffs, changelog entries, breaking changes, and summary counts.

## Type
```typescript
// src/shared/types/diff.ts
type DiffData = {
  fromVersion: string;
  toVersion: string;
  groups: {
    api: GroupDiff;
    admin: GroupDiff;
  };
};
```

## Source
- **Build-time**: Generated by `scripts/generate-diffs.mjs` using `oasdiff`
- **Runtime**: Fetched from `/data/diffs/{from}_{to}.json` via `fetchDiff(from, to)`
- **Hook**: `useDiff(from, to)` — returns `AsyncState<DiffData>`, idle when params are null

## Windowed Pairs
Not all version combinations have diffs. Each version is diffed against up to `WINDOW_SIZE=3` previous versions. The valid pairs are listed in `VersionManifest.availableDiffPairs`.

## Business Rules
- Groups are always `api` and `admin` — corresponding to the two spec files
- Diff pair key format: `"{fromVersion}_{toVersion}"` (e.g., `"v0.8.0_v0.9.0"`)
- Auto-redirect: Diff page defaults to the latest consecutive pair when no URL params
